# Business OS

고객 관리, 문의, 파일 전달, 예약과 알림을 하나의 흐름으로 연결하는 비즈니스 운영 자동화 시스템입니다.

현재는 MVP 범위와 재사용 가능한 모듈 구조를 정의하는 단계입니다.
